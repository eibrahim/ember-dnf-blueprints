# Ember-dnf-blueprints
A set of blueprints and helpers for rapid web app development.

## Installation
`ember install ember-dnf-blueprints`

## Generate a navbar
`ember g dnf-navbar main`

## Inculde bootstrap files
`ember g dnf-bootstrap install`
